# flvtag #

Tool for tagging flv files

Add following values to metadata:

 * audiocodecid
 * audiodatarate
 * audiodelay
 * audiosamplerate
 * audiosamplesize
 * audiosize
 * canSeekToEnd
 * cuePoints
 * duration
 * framerate
 * hasAudio
 * hasCuePoints
 * hasKeyframes
 * hasVideo
 * height
 * lastkeyframetimestamp
 * lasttimestamp
 * stereo
 * videocodecid
 * videodatarate
 * videosize
 * width


Following records are recalculated after building first metadata tag:

 * datasize
 * filesize
 * hasMetadata
 * keyframes
